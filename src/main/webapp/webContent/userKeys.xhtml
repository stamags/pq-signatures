Προσωπικά κλειδιά υπογραφής

Για να υπογράφετε έγγραφα με το δικό σας identity, δημιουργήστε πρώτα τα προσωπικά σας κλειδιά. Θα χρειαστείτε τον κωδικό κάθε φορά που θα υπογράφετε (ή θα τον εισάγετε μία φορά ανά σύνδεση).

Για να υπογράφετε, εισάγετε τον κωδικό keystore που ορίσατε κατά τη δημιουργία κλειδιών. Ο κωδικός θα ισχύει μέχρι να κλείσετε το browser ή να κάνετε logout.

Έχετε ήδη δημιουργήσει προσωπικά κλειδιά.

Φάκελος κλειδιών στο server:

#{userKeysBean.keysFolderPath}
Ο κωδικός keystore δεν μπορεί να ανακτηθεί αν τον ξεχάσετε. Σε αυτή την περίπτωση μπορείτε να διαγράψετε τα κλειδιά και να δημιουργήσετε νέα (οι υπογραφές που έχετε ήδη κάνει δεν θα επαληθεύονται).
